# Heartland Village General Contracting & Handyman Services ## Website — Static HTML + Tailwind CDN Same structure as Master Auto Mechanic. Pure HTML files, no build step required. --- ### 🚀 Deploy to GitHub Pages 1. Create a new GitHub repo: `heartland-village-site` 2. Upload all files maintaining this folder structure 3. Settings → Pages → Source: **GitHub Actions** 4. Push to `main` — the deploy workflow handles the rest 5.
